Scope & Topics

The International Conference on Network of the Future (NoF) covers a broad range of topics related to future internet architectures, network softwarisation, and intelligent communication systems.

Core Topics

Future Internet Architectures Clean-slate designs, information-centric networking, name-based routing
Network Softwarisation Software-Defined Networking (SDN), Network Function Virtualisation (NFV), programmable data planes
Network Slicing End-to-end slicing, resource isolation, slice orchestration
Edge & Fog Computing Edge node architectures, computation offloading, edge-to-cloud continuum
Cloud & Data Center Networks Data center topologies, virtual network embedding, resource allocation
Content Delivery CDN optimisation, adaptive streaming, edge caching strategies

Cross-Cutting Areas

Network Resilience Survivability, fault tolerance, disaster recovery
Automation & Orchestration Zero-touch operations, intent-based networking, closed-loop automation
Monitoring & Analytics Telemetry, traffic analysis, anomaly detection
ML for Networks AI-driven optimisation, predictive resource management, intelligent routing
5G/6G Networks Next-generation mobile architectures, massive MIMO, mmWave communications
Security & Trust Secure network architectures, trust management, privacy-preserving mechanisms

Publication

All accepted and presented papers are published in the NoF Conference Proceedings through IEEE Xplore. The conference is technically co-sponsored by the IEEE Communications Society.
